
Requirements:-

- Python 3.5+
- Zookeeper 3.4+

Commands:-

```
[sudo] pip install -r requirements.txt
```
Master:-
```
cd server/master
python3 master_server.py
```
Volume:-
```
cd server/volumn
python3 volumn_server.py
```

Client:-
```
cd client
python3 client.py
```

Access:-

```
upload path/to/file

download filename

ls
```
